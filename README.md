## How to Play

1. Open `index.html` in your web browser
2. Enter player names for X and O (optional)
3. Click on any cell to make your move
4. Players alternate turns
5. First player to get 3 in a row (horizontally, vertically, or diagonally) wins!
6. Use **Undo** to take back a move, **Redo** to replay it, or **Restart** for a new game
